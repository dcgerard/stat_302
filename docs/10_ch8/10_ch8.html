Linear Model Assumptions and Transformations

Learning Objectives

  • Understand the assumptions of simple linear regression and their relative importance.
  • Evaluate assumptions using scatterplots and residual plots.
  • Choose and apply appropriate remedies when assumptions are violated.
  • Interpret log-transformations of the explanatory and/or response variable.
  • Use sandwich standard errors to handle non-constant variance.
  • Perform a lack-of-fit \(F\)-test to assess whether a linear model is adequate.
  • Ch 8 in the book.

Assumptions of Simple Linear Regression

Assumptions in Decreasing Order of Importance

  1. Linearity — Does the relationship look like a straight line?
  2. Independence — Knowledge of one observation gives no information about another.
  3. Equal Variance — The spread is the same for every value of \(x\).
  4. Normality — The distribution is not too skewed and there are no extreme outliers. (Only a concern when outliers are present and the sample size is small, thanks to the CLT.)

Problems When Violated

  1. Linearity — The regression line does not capture the actual relationship.
  2. Independence — The regression line is unbiased, but standard errors are wrong.
  3. Equal Variance — The regression line is unbiased, but standard errors are wrong.
  4. Normality — Unstable results when outliers are present and the sample size is small.

Assessment Tools: Scatterplots and Residual Plots

  • Make a scatterplot of \(x\) (horizontal) vs \(y\) (vertical) to check for non-linearity, unequal variance, and normality violations.
  • A residual plot (residuals on the \(y\)-axis, fitted values on the \(x\)-axis) is often clearer because the linear signal is removed.

Demonstrating the Assumptions

Dataset 1: Gold Standard

Generate some fake data:

x <- rnorm(100, sd = 1)
y <- x + rnorm(100)
df <- tibble(x = x, y = y)
ggplot(df, aes(x = x, y = y)) +
  geom_point() +
  geom_smooth(se = FALSE)
lmout <- lm(y ~ x, data = df)
augment(lmout) |>
  ggplot(aes(x = .fitted, y = .resid)) +
  geom_point() +
  geom_hline(yintercept = 0) +
  labs(x = "Fitted values", y = "Residuals")
  • The mean lies on a straight line.
  • Residuals are centered at 0 for all \(x\).
  • Variance looks equal for all \(x\).
  • Everything looks ideal.

The distribution of x doesn’t matter. The below plot is also perfect

Dataset 2: Curved Monotone Relationship, Equal Variances

Fake data

x <- rexp(100)
x <- x - min(x) + 0.5
y <- log(x) * 20 + rnorm(100, sd = 4)
df <- tibble(x = x, y = y)
ggplot(df, aes(x = x, y = y)) +
  geom_point() +
  geom_smooth(se = FALSE)

augment(lmout) |>
  ggplot(aes(x = .fitted, y = .resid)) +
  geom_point() +
  geom_hline(yintercept = 0) +
  labs(x = "Fitted values", y = "Residuals")
  • Curved but always-increasing relationship between \(x\) and \(y\).
  • Variance looks equal for all \(x\).
  • Residual plot has a parabolic shape — a \(\log\) transformation of \(x\) could help.
df |>
  mutate(x_log = log(x)) ->
  df
ggplot(df, aes(x = x_log, y = y)) +
  geom_point() +
  geom_smooth(se = FALSE) +
  labs(x = "log(x)")
lmout <- lm(y ~ x_log, data = df)
augment(lmout) |>
  ggplot(aes(x = .fitted, y = .resid)) +
  geom_point() +
  geom_hline(yintercept = 0) +
  labs(x = "Fitted values", y = "Residuals")

After the \(\log\) transformation of \(x\), the residual plot looks flat and even — the linearity and equal-variance assumptions are now satisfied.

Dataset 3: Curved Non-Monotone Relationship, Equal Variances

Fake data

x <- rnorm(100)
y <- -x^2 + rnorm(100)
df <- tibble(x = x, y = y)
ggplot(df, aes(x = x, y = y)) +
  geom_point() +
  geom_smooth(se = FALSE)
lmout <- lm(y ~ x, data = df)
augment(lmout) |>
  ggplot(aes(x = .fitted, y = .resid)) +
  geom_point() +
  geom_hline(yintercept = 0) +
  labs(x = "Fitted values", y = "Residuals")
  • Curved relationship that sometimes increases, sometimes decreases.
  • Variance looks equal for all \(x\).
  • Residual plot has a parabolic form.

Solution 1: Add a quadratic term: \[Y_i = \beta_0 + \beta_1 X_i + \beta_2 X_i^2\]

quad_lm <- lm(y ~ x + I(x^2), data = df)
df |>
  mutate(fitted = fitted(quad_lm)) ->
  df
ggplot(df, aes(x = x, y = y)) +
  geom_point() +
  geom_line(aes(x = x, y = fitted), color = "blue", linewidth = 1)
augment(quad_lm) |>
  ggplot(aes(x = .fitted, y = .resid)) +
  geom_point() +
  geom_hline(yintercept = 0) +
  labs(x = "Fitted values", y = "Residuals")

Solution 2: Fit a broken-line (piecewise linear) model using the lm.br package: \[Y_i = \begin{cases} \beta_0 + \beta_1 X_i & \text{if } X_i < C \\ \beta_0^* + \beta_1^* X_i & \text{if } X_i > C \end{cases}\]

lmbr_out <- lm.br(y ~ x, data = df)
df |>
  mutate(fitted = fitted(lmbr_out)) ->
  df
ggplot(df, aes(x = x, y = y)) +
  geom_point() +
  geom_line(aes(x = x, y = fitted), color = "blue", linewidth = 1)
tibble(.fitted = fitted(lmbr_out), .resid = resid(lmbr_out)) |>
  ggplot(aes(x = .fitted, y = .resid)) +
  geom_point() +
  geom_hline(yintercept = 0) +
  labs(x = "Fitted values", y = "Residuals")

Dataset 5: Linear Relationship, Equal Variances, Skewed Errors

Fake data

x <- runif(200)
y <- 15 * x + rexp(200, 0.2)
df <- tibble(x = x, y = y)
ggplot(df, aes(x = x, y = y)) +
  geom_point() +
  geom_smooth(se = FALSE)
lmout <- lm(y ~ x, data = df)
augment(lmout) |>
  ggplot(aes(x = .fitted, y = .resid)) +
  geom_point() +
  geom_hline(yintercept = 0) +
  labs(x = "Fitted values", y = "Residuals")
  • Straight-line relationship between \(x\) and \(y\).
  • Variances are approximately equal for all \(x\).
  • The errors are right-skewed — visible in the residual plot.

Solutions:

  1. Do nothing, but report the skewness (usually acceptable for large \(n\)).
  2. Fit a quantile regression for the median: \[\text{Median}(Y_i) = \beta_0 + \beta_1 X_i\]
qr_out <- rq(y ~ x, tau = 0.5, data = df)

df |>
  mutate(qr_fitted = fitted(qr_out)) ->
  df
ggplot(df, aes(x = x, y = y)) +
  geom_point() +
  geom_smooth(method = "lm", se = FALSE) +
  geom_line(aes(x = x, y = qr_fitted), color = "red", linewidth = 1)

The quantile regression line (red) is not very different from the OLS line (blue) — skewness with equal variance has little practical effect on the coefficient estimates.

Intuition of Sandwich Estimator of Variance

Simplified model: \(Y_i = \beta_1 X_i\) (zero intercept).

Using calculus: \[\hat{\beta}_1 = \frac{\sum_{i=1}^n X_i Y_i}{\sum_{i=1}^n X_i^2}\]

So: \[\text{Var}(\hat{\beta}_1) = \text{Var}\!\left(\frac{\sum_{i=1}^n X_i Y_i}{\sum_{i=1}^n X_i^2}\right) = \frac{\sum_{i=1}^n X_i^2\,\text{Var}(Y_i \mid X_i)}{\left(\sum_{i=1}^n X_i^2\right)^2}\]

  • Usual method: Estimate \(\text{Var}(Y_i \mid X_i)\) with \(s_p^2\) — assumes the same variance for all \(i\).
  • Sandwich method: Estimate \(\text{Var}(Y_i \mid X_i)\) with \((Y_i - \hat{\beta}_1 X_i)^2\) — allows a different variance at each \(i\).

The sandwich estimator yields accurate standard errors for the coefficient estimates as long as:

  1. The linearity assumption holds, and
  2. The sample size is large enough.

Note that sandwich standard errors cannot be used for prediction intervals.

Interpreting Log Transformations

No Log on \(X\), No Log on \(Y\)

Model: \(\mu(Y \mid X) = \beta_0 + \beta_1 X\)

At \(X + 1\): \[\mu(Y \mid X+1) = \beta_0 + \beta_1(X+1) = \mu(Y \mid X) + \beta_1\]

  • Observational study: \(\beta_1\) is the mean difference in \(Y\) between groups that differ by one unit in \(X\).
  • Randomized experiment: \(\beta_1\) is the mean increase in \(Y\) when \(X\) is increased by one unit.

No Log on \(X\), Log on \(Y\)

Model: \(\mu(\log(Y) \mid X) = \beta_0 + \beta_1 X\)

If residuals are fairly symmetric, then \(\text{Median}(\log Y \mid X) \approx \mu(\log Y \mid X)\), so:

\[\text{Median}(Y \mid X) = \exp(\beta_0 + \beta_1 X) = \exp(\beta_0)\exp(\beta_1 X)\]

At \(X + 1\): \[\text{Median}(Y \mid X+1) = \text{Median}(Y \mid X) \cdot \exp(\beta_1)\]

  • Observational study: \(\exp(\beta_1)\) is the ratio of medians when groups differ by one unit in \(X\).
  • Randomized experiment: \(\exp(\beta_1)\) is the multiplicative change in median \(Y\) when \(X\) increases by one unit.

Voltage and Breakdown Time):

  geom_point() +

  scale_y_log10()

  • We fit the model

    lm(I(log(Time)) ~ Voltage, data = voltage) |>
  tidy()
    # A tibble: 2 × 5
  term        estimate std.error statistic  p.value
  <chr>          <dbl>     <dbl>     <dbl>    <dbl>
1 (Intercept)   19.0      1.91        9.92 3.05e-15
2 Voltage       -0.507    0.0574     -8.84 3.34e-13

  • \(\hat\beta_1 = -0.51\),

  • 95% CI \((-0.62, -0.39)\).

  • \(\exp(-0.51) = 0.6\): a 1 kV increase is associated with 60% of the median breakdown time. So a 40% decrease.

    \[ \text{Median}(Y \mid X+1) - \text{Median}(Y \mid X) = (\exp(\beta_1) - 1)\,\text{Median}(Y \mid X) = -0.4\,\text{Median}(Y \mid X) \]

  • 95% CI for the multiplicative effect: \((\exp(-0.62), \exp(-0.39)) = (0.54, 0.68)\). So a a 32% to 46% decrease.

Log on \(X\), No Log on \(Y\)

Model: \(\mu(Y \mid X) = \beta_0 + \beta_1 \log(X)\)

When \(X\) is doubled:

\[\mu(Y \mid 2X) = \beta_0 + \beta_1\log(2X) = \mu(Y \mid X) + \beta_1\log(2)\]

  • Observational study: \(\beta_1\log(2)\) is the mean difference in \(Y\) when one group’s \(X\) is twice the other’s.
  • Randomized experiment: \(\beta_1\log(2)\) is the increase in mean \(Y\) when \(X\) is doubled.

A similar interpretation holds for \(\beta_1\log(10)\) and a 10-fold increase in \(X\).

Log on \(X\), Log on \(Y\)

Model: \(\mu(\log(Y) \mid X) = \beta_0 + \beta_1\log(X)\)

Simplifying: \[\text{Median}(Y \mid X) = \exp(\beta_0)\,X^{\beta_1}\]

When \(X\) is doubled: \[\text{Median}(Y \mid 2X) = \text{Median}(Y \mid X) \cdot 2^{\beta_1}\]

  • Observational study: \(2^{\beta_1}\) is the ratio of medians when one group’s \(X\) is twice the other’s.
  • Randomized experiment: \(2^{\beta_1}\) is the multiplicative change in median \(Y\) when \(X\) is doubled.

A similar interpretation holds for \(10^{\beta_1}\) and a 10-fold increase in \(X\).

Other Transformations

Other transformations (e.g., \(1/Y\), \(\sqrt{Y}\)) may improve residual plots but lack clean interpretations. They are appropriate when the goal is:

  1. Prediction, or
  2. Only detecting whether an association exists (not describing its size).
  • \(1/Y\) tends to fix more extreme non-constant variance.
  • \(\sqrt{Y}\) tends to fix less extreme non-constant variance.

Which Is Better?

  • If the linear model appears to fit fine, it is always preferred.
  • You can interpolate with the linear model (not with ANOVA).
  • The linear model has easier interpretations.
  • The linear model has fewer parameters.
  • We can formally test if the linear model does not fit using the \(F\)-testing strategy if we have replicates at given values of \(X\).

Lack-of-Fit \(F\)-Test

  • \(H_0: E[Y_i] = \beta_0 + \beta_1X_i\) (mean is based on line)

  • \(H_A: E[Y_i] = \mu_j\) where \(X_i = j\) (mean is based on group)

  case0802
rss_full    <- sum((case0802$logTime - case0802$mean)^2)
rss_reduced <- sum((case0802$logTime - case0802$fitted)^2)
ESS         <- rss_reduced - rss_full
df_full     <- 69
df_reduced  <- 74
df_extra    <- df_reduced - df_full
fstat       <- (ESS / df_extra) / (rss_full / df_full)

  • \(RSS_{full}\) = 173.7489206

  • \(df_{full} = n - I\) = 76 - 7 = 69

  • \(RSS_{reduced}\) = 180.0748361

  • \(df_{reduced} = n - 2\) = 76 - 2 = 74

  • \(ESS = RSS_{reduced} - RSS_{full}\) = 6.3259155

  • \(df_{extra} = df_{reduced} - df_{full}\) = 74 - 69 = 5

  • \(F\)-statistic \(= \frac{ESS / df_{extra}}{RSS_{full} / df_{full}}\) = 0.5024356
