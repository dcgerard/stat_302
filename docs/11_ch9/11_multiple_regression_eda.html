Case Study: Brain Size

  • What variables are associated with brain weight?
  • Collected information on 96 different species.
  • We know that body weight is already associated with brain weight,
    • So what variables are associated with brain weight after controlling for body weight?
  • Possible variables: Body weight (kg), gestation period (days), litter size.

Plotting the Response Against Each Explanatory Variable

  • The first step is almost always making a ton of scatterplots.
  • Plots of the response against each explanatory variable show what variables seem to be marginally related to the response.
  • “Marginally related” = related unconditionally on other variables.
ggplot(case0902, aes(x = Body, y = Brain)) +
  geom_point()
ggplot(case0902, aes(x = Gestation, y = Brain)) +
  geom_point()
ggplot(case0902, aes(x = Litter, y = Brain)) +
  geom_point()

Curvature and different spreads at each explanatory variable suggest a log transformation of Brain.

case0902$logBrain <- log(case0902$Brain)
ggplot(case0902, aes(x = Body, y = logBrain)) +
  geom_point()
ggplot(case0902, aes(x = Gestation, y = logBrain)) +
  geom_point()
ggplot(case0902, aes(x = Litter, y = logBrain)) +
  geom_point()

Still a lot of curvature, so it looks like logging each variable might help.

case0902$logBody      <- log(case0902$Body)
case0902$logGestation <- log(case0902$Gestation)
case0902$logLitter    <- log(case0902$Litter)
ggplot(case0902, aes(x = logBody, y = logBrain)) +
  geom_point()
ggplot(case0902, aes(x = logGestation, y = logBrain)) +
  geom_point()
ggplot(case0902, aes(x = logLitter, y = logBrain)) +
  geom_point()

There looks like there is a lot of linearity now.

Plotting Explanatory Variables Against Each Other

  • It is often useful to look at scatterplots between each pair of explanatory variables.
  • This tells us if some variables seem to be picking up a lot of the same information.
  • E.g., gestation period might be larger just because body size is larger.
    • If brain is associated with gestation, it might only be through body weight.
ggplot(case0902, aes(x = logBody, y = logGestation)) +
  geom_point()
ggplot(case0902, aes(x = logBody, y = logLitter)) +
  geom_point()
ggplot(case0902, aes(x = logGestation, y = logLitter)) +
  geom_point()

Matrix Plot

  • You can show scatterplots between all variables at the same time.
  • This is called a matrix plot (or a pairs plot).
ggpairs(case0902, columns = 6:9)

Initial Model Fit

  • We can first fit a very complicated model and check residuals.
  • We would be looking for more curvature and outliers.
lm_comp <- lm(logBrain ~ logBody * logLitter * logGestation,
              data = case0902)
res_vec <- resid(lm_comp)
fit_vec <- fitted(lm_comp)
ggplot(data.frame(fit_vec, res_vec), aes(x = fit_vec, y = res_vec)) +
  geom_point() +
  geom_hline(yintercept = 0) +
  labs(x = "Fitted values", y = "Residuals")
ggplot(data.frame(res_vec), aes(sample = res_vec)) +
  geom_qq() +
  geom_qq_line()

Coded Scatterplots

Quantitative Third Variable

  • If you want to explore the association between three quantitative variables, you can code one of them by transparency (more preferable) or size (less preferable).
ggplot(case0902, aes(x = logBody, y = logBrain, alpha = logGestation)) +
  geom_point()
ggplot(case0902, aes(x = logBody, y = logBrain, size = logGestation)) +
  geom_point()

Categorical Third Variable

  • If you have categorical explanatory variables, you can code their levels by colors (more preferable) and shapes (less preferable) and include this on a scatterplot of two quantitative variables.
case0901 <- read_csv("https://dcgerard.github.io/stat_302/data/case0901.csv")
case0901$Time <- as.factor(case0901$Time)
ggplot(case0901, aes(x = Intensity, y = Flowers, color = Time)) +
  geom_point()
ggplot(case0901, aes(x = Intensity, y = Flowers, shape = Time)) +
  geom_point()

It’s polite to use colorblind-safe color palettes. The Okabe-Ito is one colorblind-safe palette.

ggplot(case0901, aes(x = Intensity, y = Flowers, color = Time)) +
  geom_point() +
  scale_color_discrete(palette = "Okabe-Ito")
