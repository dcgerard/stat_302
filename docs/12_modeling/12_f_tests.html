F-tests for Nested Models

Author

David Gerard

Published

June 24, 2026

Learning Objectives

  • Test whether a group of explanatory variables jointly improves a model.
  • Distinguish nested from non-nested model pairs.
  • Interpret the \(F\)-statistic and its degrees of freedom.
  • Use anova() in R to run the \(F\)-test.
  • Section 10.3 in the textbook.

Motivation

The Problem with Individual \(t\)-tests

  • In multiple regression we can test each coefficient with a \(t\)-test.
  • But some variables enter the model through multiple terms (e.g., \(X\) and \(X^2\)).
  • Testing \(\beta_1 = 0\) and \(\beta_2 = 0\) separately does not answer “does \(X\) matter at all?”
  • We need to test \(H_0: \beta_1 = 0\) and \(\beta_2 = 0\) simultaneously — a joint test.

Case Study: Kentucky Derby

  • The Kentucky Derby is a famous horse race held annually since 1875.
  • We have the winning speed (mph) and the year of each race.
  • Does year explain speed? (Speed has increased over time as breeding and track conditions improved.)

ggplot(ex0920, aes(x = Year, y = Speed)) +

  • The trend is clearly nonlinear: speed rose sharply in the early 20th century then leveled off.
  • A quadratic model is appropriate: \[\mu(\text{Speed} \mid \text{Year}) = \beta_0 + \beta_1\,\text{Year} + \beta_2\,\text{Year}^2\]
  • Question: Is year at all associated with winning speed?
  • \(H_0: \beta_1 = 0\) and \(\beta_2 = 0\)
  • \(H_A: \beta_1 \neq 0\) or \(\beta_2 \neq 0\)

Nested Models

Definition

  • A model is nested inside another if it can be obtained by setting some coefficients in the larger model to zero.
  • The larger model is the full model.
  • The smaller model is the reduced model.

Kentucky Derby Example

Model Equation
Full \(\mu(\text{Speed} \mid \text{Year}) = \beta_0 + \beta_1\,\text{Year} + \beta_2\,\text{Year}^2\)
Reduced (under \(H_0\)) \(\mu(\text{Speed}) = \beta_0\)
  • The reduced model is the full model with \(\beta_1 = \beta_2 = 0\).

Non-Nested Models (Counter-example)

  • Model A: \(\mu(\text{Speed} \mid \text{Year}) = \beta_0 + \beta_1\,\text{Year} + \beta_2\,\text{Year}^2\)
  • Model B: \(\mu(\text{Speed} \mid \text{Starters}) = \beta_0 + \beta_1\,\text{Starters} + \beta_2\,\text{Starters}^2\)
  • Neither is a special case of the other — cannot use an \(F\)-test.
  • For non-nested models, use AIC or BIC (see the model selection lecture).

Intuition: Residuals Tell the Story

  • If the full model fits much better (smaller residuals), we have evidence against \(H_0\).
  • We compare the residual sum of squares (RSS) under each model.
ex0920 <- ex0920 |> mutate(Year2 = Year^2)
lm_full    <- lm(Speed ~ Year + Year2, data = ex0920)
lm_reduced <- lm(Speed ~ 1,            data = ex0920)

ex0920 <- ex0920 |>
  mutate(fit_full = fitted(lm_full),
         fit_red  = fitted(lm_reduced))
  • The full model fits much better — the residuals are far smaller.
  • This large difference in fit is strong evidence against \(H_0\).

The \(F\)-Statistic

Key Quantities

Let \(p_F\) = number of parameters in the full model, \(p_R\) = number in the reduced, \(n\) = sample size.

Symbol Meaning
\(RSS_F\) Sum of squared residuals under the full model
\(RSS_R\) Sum of squared residuals under the reduced model
\(ESS\) Extra sum of squares \(= RSS_R - RSS_F\) (how much better the full fits)
\(df_{\text{extra}}\) \(p_F - p_R\) (extra parameters in the full model)
\(df_F\) \(n - p_F\) (residual degrees of freedom for the full model)

Formula and Distribution

\[F = \frac{ESS / \, df_{\text{extra}}}{RSS_F / \, df_F}\]

  • Numerator: average extra variability explained per additional parameter.
  • Denominator: average residual variability per degree of freedom (an estimate of \(\sigma^2\)).
  • Under \(H_0\): \(F\) follows an \(F\)-distribution with \((df_{\text{extra}},\, df_F)\) degrees of freedom.
  • A large \(F\) means the full model explains substantially more — evidence against \(H_0\).

Running the \(F\)-Test in R

Use anova() with the reduced model first, the full model second:

anova(lm_reduced, lm_full)
Analysis of Variance Table

Model 1: Speed ~ 1
Model 2: Speed ~ Year + Year2
  Res.Df  RSS Df Sum of Sq   F Pr(>F)
1    115 93.0                        
2    113 33.1  2      59.9 102 <2e-16

Reading the Output

Column Meaning
Res.Df Residual degrees of freedom for each model
RSS Residual sum of squares for each model
Df \(df_{\text{extra}} = p_F - p_R\)
Sum of Sq \(ESS = RSS_R - RSS_F\)
F \(F\)-statistic
Pr(>F) \(p\)-value
  • Conclusion: We have overwhelming evidence that year is associated with winning speed (\(F = 104.6\) on 2 and 119 df, \(p < 0.001\)).

Another Example: Do Starters Matter?

Setup

  • Starters: number of horses that started the race.
  • After accounting for a quadratic trend in year, does the number of starters add anything?

\[H_0: \beta_3 = \beta_4 = 0 \text{ in:}\] \[\mu(\text{Speed} \mid \text{Year}, \text{Starters}) = \beta_0 + \beta_1\,\text{Year} + \beta_2\,\text{Year}^2 + \beta_3\,\text{Starters} + \beta_4\,\text{Starters}^2\]

  • Full: Year + Year² + Starters + Starters²
  • Reduced: Year + Year² (same as the full model above)
ex0920 <- ex0920 |> mutate(Starters2 = Starters^2)
lm_full2    <- lm(Speed ~ Year + Year2 + Starters + Starters2, data = ex0920)
lm_reduced2 <- lm(Speed ~ Year + Year2,                        data = ex0920)
anova(lm_reduced2, lm_full2)
Analysis of Variance Table

Model 1: Speed ~ Year + Year2
Model 2: Speed ~ Year + Year2 + Starters + Starters2
  Res.Df  RSS Df Sum of Sq    F Pr(>F)
1    113 33.1                         
2    111 30.9  2      2.18 3.92  0.023
  • Conclusion: We do have some evidence that the number of starters is associated with speed after adjusting for year (\(F = 3.92\) on 2 and 111 df, \(p = 0.023\)).
