Learning Objectives

  • Write out the additive model and the interaction model.

  • Test for main effects and interactions using F-tests.
  • Obtain coefficient estimates and confidence intervals using tidy().

Case Study: Pygmalion Effect

  • Pygmalion Effect: high expectations from a supervisor translate into improved performance of a subordinate.
  • A study used U.S. Army companies, each containing several platoons.
  • Within each company, one platoon was randomly designated the “Pygmalion platoon.”
  • The commanding officer of the Pygmalion platoon was told (falsely) that his platoon had been predicted to be superior soldiers.
  • At the end of basic training, all soldiers took a skill test.
  • Data: average score for each platoon. Company is a nuisance variable (grouping variable).

When to Use Two-Way ANOVA

  1. Quantitative response variable.
  2. Two categorical explanatory variables.

The Models

The Additive Model (Main Effects Only)

\[Y_{ijk} = \mu + \alpha_i + \beta_j + \epsilon_{ijk}\]

  • \(\mu\): baseline mean (Company C1, Control group).
  • \(\alpha_i\): main effect of Company \(i\) — accounts for between-company variability.
  • \(\beta_j\): main effect of Treatment \(j\) — what we actually care about.
  • \(\epsilon_{ijk} \sim N(0, \sigma^2)\): noise.
  • Additivity assumption: the treatment effect \(\beta_j\) is the same for every company.

It is called two-way because each observational unit can be placed into a two-way table according to its group in both categorical variables.

C1 C2 \(\cdots\) C10
Pygmalion \(\mu + \alpha_1 + \beta_1\) \(\mu + \alpha_2 + \beta_1\) \(\cdots\) \(\mu + \alpha_{10} + \beta_1\)
Control \(\mu + \alpha_1 + \beta_2\) \(\mu + \alpha_2 + \beta_2\) \(\cdots\) \(\mu + \alpha_{10} + \beta_2\)
  • In the Pygmalion case, we are not actually interested in the \(\alpha_i\)’s (company effects).
    • The researchers were using this as a grouping variable to decrease variance.

Testing for Main Effects

The Research Question

  • Research question: Does the Pygmalion treatment improve scores?
  • We need to test \(H_0: \beta_j = 0\) for all \(j\) (treatment has no effect).
  • Equivalently: does the full additive model fit better than the reduced model (Company only)?

\[\text{Reduced: } Y_{ijk} = \mu + \alpha_i + \epsilon_{ijk}\] \[\text{Full: } Y_{ijk} = \mu + \alpha_i + \beta_j + \epsilon_{ijk}\]

Reduced Model: Company Only

  • Under the reduced model, each company has one estimated mean for all platoons.

The residuals under the reduced model — gaps between points and their company mean:

Full (Additive) Model: Company + Treatment

  • Under the additive model, each company gets two estimated means — one per treatment.
  • The treatment gap is the same for every company (additivity).

The residuals under the additive model are much smaller — the systematic treatment pattern has been absorbed by \(\beta_j\):

Estimated Treatment Effects

The arrows show the Control → Pygmalion gap at each company’s estimated means. Under the additive model, all arrows are forced to be the same length — this is exactly what \(\hat{\beta}_j\) being a single constant means:

The F-Statistic

Quantity Formula Value
\(RSS_{\text{full}}\) 778.5
\(df_{\text{full}}\) \(n - I - J + 1 = 29 - 10 - 2 + 1\) 18
\(RSS_{\text{reduced}}\) 1117.39
\(df_{\text{reduced}}\) \(n - I = 29 - 10\) 19

\[ESS = RSS_{\text{reduced}} - RSS_{\text{full}} = 1117.39 - 778.5 = 338.88\]

\[df_{\text{extra}} = df_{\text{reduced}} - df_{\text{full}} = 19 - 18 = 1\]

\[F = \frac{ESS / \, df_{\text{extra}}}{RSS_{\text{full}} / \, df_{\text{full}}} = \frac{338.88 / 1}{778.5 / 18} = 7.84\]

Compare the observed \(F\) to an \(F_{1, 18}\) distribution — the shaded area is the \(p\)-value:

F-Test in R

aout_add <- aov(Score ~ Company + Treat, data = case1302)
tidy(aout_add)

  • The Treat row is the F-test for the main effect of treatment controlling for Company.

Testing for Interactions

But Wait — Is the Additivity Assumption Valid?

  • The test above assumes the treatment effect is the same for every company.
  • What if the effect of the Pygmalion treatment depends on which company?
  • We should check this assumption — i.e., test for interactions.

The Interaction Model (Cell Means)

\[Y_{ijk} = \mu + \alpha_i + \beta_j + (\alpha\beta)_{ij} + \epsilon_{ijk}\]

  • \((\alpha\beta)_{ij}\): interaction — allows the treatment effect to differ by company.
    • This is not multipliciation. \((\alpha\beta)_{ij}\) is a single number, not the product of two numbers.
  • Equivalent to letting every company × treatment combination have its own mean.
  • In R: Score ~ Company * Treat.
C1 C2 \(\cdots\) C10
Pygmalion \(\mu + \alpha_1 + \beta_1 + (\alpha\beta)_{11}\) \(\mu + \alpha_2 + \beta_1 + (\alpha\beta)_{21}\) \(\cdots\) \(\mu + \alpha_{10} + \beta_1 + (\alpha\beta)_{10,1}\)
Control \(\mu + \alpha_1 + \beta_2 + (\alpha\beta)_{12}\) \(\mu + \alpha_2 + \beta_2 + (\alpha\beta)_{22}\) \(\cdots\) \(\mu + \alpha_{10} + \beta_2 + (\alpha\beta)_{10,2}\)
  • The cell means model is the exact same as a one-way ANOVA model with each cell having its own mean:

\[ Y_{\ell k} = \mu + \alpha_{\ell} + \epsilon_{\ell k} \]

C1 C2 \(\cdots\) C10
Pygmalion \(\mu + \alpha_{1}\) \(\mu + \alpha_{2}\) \(\cdots\) \(\mu + \alpha_{10}\)
Control \(\mu + \alpha_{11}\) \(\mu + \alpha_{12}\) \(\cdots\) \(\mu + \alpha_{20}\)

Interaction Plots

  • An interaction plot shows the group means connected by lines.
  • X-axis: levels of one variable. Color: levels of the other variable.
  • Parallel lines \(\Rightarrow\) additive model is plausible.
  • Non-parallel lines \(\Rightarrow\) interaction may be present.

Interpreting the Interaction Model

Under the cell means model, each company × treatment combination gets its own estimated mean:

The arrows make the company-specific treatment effects explicit — each company gets its own arrow length:

  • The residuals under the cell means model are smaller than under the additive model — each company’s idiosyncratic effect is absorbed:

    • Recall that the full model will always have smaller residuals on average. We need to test if these are smaller than what would be expected by chance alone.

Hypotheses

\[H_0: (\alpha\beta)_{ij} = 0 \text{ for all } i, j\] \[H_A: \text{at least one } (\alpha\beta)_{ij} \neq 0\]

Degrees of Freedom

Model Parameters Residual df
Full (cell means, with interaction) \(I \times J\) \(n - IJ\)
Reduced (additive, no interaction) \(I + J - 1\) \(n - I - J + 1\)
Extra \(IJ - I - J + 1\) —

The F-Statistic

Quantity Formula Value
\(RSS_{\text{full}}\) (cell means) 467.04
\(df_{\text{full}}\) \(n - IJ = 29 - 10 \times 2\) 9
\(RSS_{\text{reduced}}\) (additive) 778.5
\(df_{\text{reduced}}\) \(n - I - J + 1 = 29 - 10 - 2 + 1\) 18

\[ESS = RSS_{\text{reduced}} - RSS_{\text{full}} = 778.5 - 467.04 = 311.46\]

\[df_{\text{extra}} = df_{\text{reduced}} - df_{\text{full}} = 18 - 9 = 9\]

\[F = \frac{ESS / \, df_{\text{extra}}}{RSS_{\text{full}} / \, df_{\text{full}}} = \frac{311.46 / 9}{467.04 / 9} = 0.67\]

Compare the observed \(F\) to an \(F_{9, 9}\) distribution:

F-Test in R

R Formula Syntax

Formula Model fitted
A + B \(\mu + \alpha_i + \beta_j\) (additive)
A * B \(\mu + \alpha_i + \beta_j + (\alpha\beta)_{ij}\) (with interaction)
A:B \((\alpha\beta)_{ij}\) only (rarely used alone)
A * B - A:B Same as A + B

Analysis Workflow

  1. Make an interaction plot to assess whether additivity is plausible.
  2. Test for interaction: if no evidence (\(p > 0.05\)), adopt the additive model.
  3. Test for main effects using the additive model.
  4. Estimate and interpret coefficients with confidence intervals.
